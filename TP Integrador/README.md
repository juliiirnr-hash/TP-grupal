

### ***Descripción de el programa***



Este programa sirve para consultar información acerca de el o los países que el usuario desee, teniendo un menú apenas comienza la ejecución donde se solicitan las diferentes acciones que el usuario puede realizar. Los datos utilizados están almacenados en un archivo CSV el cual esta cargado con la información necesaria para el funcionamiento del programa: nombre del país, población, superficie y continente.



### ***Instrucciones de uso***



Para utilizar el programa se debe instalar la carpeta "TP INTEGRADOR" (debe contener los archivos "pais.py","TP\_integrador.py" y "países.csv") desde el repositorio remoto de nuestro proyecto. Para funcionar el proyecto necesitará de un IDE (como Visual Studio Code) y tener instalada la ultima versión de Python.



Para ejecutarlo se deberá abrir la carpeta en el IDE y correrlo por su terminal (el programa es indiferente ante mayúsculas y minúsculas pero NO ante tildes ni espacios innecesarios) , se mostrará un menú que contiene las siguientes funcionalidades:



a. Buscar País

b. Filtrar países

c. Ordenar países

d. Estadísticas de países

e. Salir





##### *buscar país:*



el programa solicitará el país requerido por el usuario, una vez ingresado mostrará su población, superficie y continente.



##### *filtrar países:*



el programa mostrará las siguientes opciones:



*1. Buscar por continente;*

se solicita un continente, una vez ingresado mostrará los países por los que está conformado.

*2. Rango poblacional;*

se solicita un país, una vez ingresado mostrará los países clasificados por cantidad de población.

*3. Rango de superficie;*

se solicita un país, una vez ingresado mostrará los países clasificados por los km² de superficie.



##### *ordenar países:*



*el programa mostrará un listado ordenado alfabéticamente de los países, población(ascendente) y superficie(ascendente).*



##### *estadísticas de países:*



*el programa mostrará un promedio de población/superficie, países con menor/mayor población y cantidad de países por continentes.*



##### *salir:*



*termina la ejecución del programa.*
